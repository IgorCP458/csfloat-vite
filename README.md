## 📦 Instalação

1. **Clone o repositório**
    
    ```bash
    git clone https://github.com/IgorCP458/csfloat-vite.git
    ```
    
2. **Entre na pasta do projeto**
    
    ```bash
    cd csfloat-vite
    ```
    
3. **Instale as dependências**
    
    ```bash
    npm install
    ```
    

## 🔨 Rodando o servidor de desenvolvimento

```bash
npm run dev
```

## 🔥 Build para produção

```bash
npm run build
```
